# Brief Summary

This notebook demonstrates basic Pandas workflow steps for an ecommerce-style dataset. It loads a CSV when available, explores the dataframe, handles missing values, filters rows, selects columns, removes duplicates, creates `total_amount = price * quantity`, and saves the cleaned output as `ecommerce_dataset_cleaned.csv`.

Because the real CSV was not available in the environment, the notebook includes a small demo fallback dataset so it can still run end to end.
